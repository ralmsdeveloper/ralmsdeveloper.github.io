O Que é ASP.NET Core?

ASP.NET Core é uma estrutura de software livre de plataforma cruzada, de alto desempenho para a criação de aplicativos modernos conectados à Internet, em nuvem.

A verdade é que a Microsoft gostaria de inovar, mas sem se preocupar com código legado, que já existia desde 2002.
